== Description ==
The plugin helps you sell anything via Telegram with Woocommerce help.
For now it works with simple products only.
You can see an example here https://t.me/wootgbot

= Plugin Functionality: =
* Search for @BotFather in Telegram
* Create a new bot by sending /newbot command
* Get its token
* Go to WooCommerce / Settings / Integration / Sell via Telegram and set the token
* Your bot is ready to work!

== Installation ==

* Download the plugin
* Upload the folder "wc-telegram" to wp-content/plugins (or upload a zip through the WordPress admin)
